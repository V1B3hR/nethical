# ISMS (Information Security Management System) Artifacts

This folder contains templates and documents required for ISO/IEC 27001:2022 certification and ongoing ISMS maintenance.

## Document Index

| Document | ISO 27001 Control | Description |
|----------|-------------------|-------------|
| [Information Security Policy](./information_security_policy.md) | A.5.1 | Top-level security policy |
| [Asset Register](./asset_register.md) | A.5.9 | Inventory of information assets |
| [Change Management Policy](./change_management_policy.md) | A.8.32 | Change control procedures |
| [Acceptable Use Policy](./acceptable_use_policy.md) | A.5.10 | Acceptable use of assets |
| [Supplier Security Policy](./supplier_security_policy.md) | A.5.19 | Supplier relationship security |

## Usage

These documents are templates that should be customized for your organization's specific needs during ISO 27001 certification.

## Review Schedule

All ISMS documents should be reviewed:
- **Annually** as part of management review
- **After significant changes** to the information security environment
- **Following security incidents** that reveal policy gaps
