## 14. Glossary

**Appeal**: A formal request to reconsider a decision made by the system.

**Disparate Impact**: A measure of whether different groups experience different outcomes at disproportionate rates.

**Hash Chain**: A cryptographic data structure linking policy versions to ensure integrity and non-repudiation.

**Merkle Tree**: A tree structure where each leaf is a hash of a data block, used for efficient and secure verification of large data sets.

**Statistical Parity**: A fairness metric requiring that outcomes are distributed equally across different demographic groups.
